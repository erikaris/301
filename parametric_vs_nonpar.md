# Parametric vs Non-parametric / Ordinal / Paired Tests

| **No.** | **Purpose / Data Type**                                     | **Parametric Test (Normal / Interval)**                 | **Non-parametric / Ordinal / Robust Alternative**                                     | **R Implementation**                                                                                                            |      |
| ------- | ----------------------------------------------------------- | ------------------------------------------------------- | ------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- | ---- |
| **1**   | One-sample location (interval/ratio)                        | One-sample t-test                                       | Wilcoxon signed-rank; Sign test                                                       | `t.test(x, mu=0)`; `wilcox.test(x, mu=0)`; `BSDA::SIGN.test(x, md=0)`                                                           |      |
| **2**   | Two independent samples (interval/ratio, equal variances)   | Independent t-test (student t)                          | Mann–Whitney U (Wilcoxon rank-sum); 2-sample KS                                       | `t.test(x ~ g, var.equal=TRUE)`; `wilcox.test(x ~ g)`; `ks.test(x, y)`                                                          |      |
| **3**   | Two independent samples (interval/ratio, unequal variances) | Independent t-test (Welch’s t-test)                     | Mann–Whitney U (Wilcoxon rank-sum); 2-sample KS                                       | `t.test(x ~ g)`; `wilcox.test(x ~ g)`; `ks.test(x, y)`                                                                          |      |
| **4**   | Two paired samples (interval/ratio)                         | Paired t-test                                           | Wilcoxon signed-rank; Sign test                                                       | `t.test(x, y, paired=TRUE)`; `wilcox.test(x, y, paired=TRUE)`; `BSDA::SIGN.test(x - y)`                                         |      |
| **5**   | ≥3 independent groups (interval/ratio, equal variances)     | Welch ANOVA --> followed by **Tukey** post-hoc          | Kruskal–Wallis --> post-hoc: Dunn's Test                                              | `oneway.test(y ~ g)`; Tukey: `TukeyHSD(aov(y ~ g))`; `kruskal.test(y ~ g)`; Dunn: `FSA::dunnTest()`                             |      |
| **6**   | ≥3 independent groups (interval/ratio, unequal variances)   | One-way ANOVA --> followed by **Games-Howell** post-hoc | Kruskal–Wallis; permutation ANOVA                                                     | `aov(y ~ g)`; `rstatix::games_howell_test()`; `kruskal.test(y ~ g)`; permutation: `coin::oneway_test()`                         |      |
| **7**   | ≥3 related/repeated measures                                | Repeated-measures ANOVA                                 | Friedman test                                                                         | `aov(y ~ time + Error(id/time))`; `ez::ezANOVA()`; `friedman.test(y ~ time                                                      | id)` |
| **8**   | Two-way factorial (interval/ratio)                          | Two-way ANOVA                                           | Aligned Rank Transform (ART); Scheirer–Ray–Hare (SRH); permutation factorial ANOVA    | `aov(y ~ A * B)`; ART: `ARTool::art(y ~ A * B)`; SRH: `rcompanion::scheirerRayHare()`; permutation: `coin::independence_test()` |      |
| **9**   | ANCOVA (factor(s) + covariate)                              | ANCOVA                                                  | Quade’s test (rank ANCOVA); permutation ANCOVA; robust regression                     | `aov(y ~ group + covariate)`; `rcompanion::quadeTest()`; permutation: `coin::ancova_test()`; robust: `MASS::rlm()`              |      |
| **10**  | Correlation (interval/ratio)                                | Pearson correlation                                     | Spearman rank correlation; Kendall’s tau                                              | `cor.test(x, y, method="pearson")`; `"spearman"`; `"kendall"`                                                                   |      |
| **11**  | Regression coefficients (interval/ratio)                    | t-test for slope (OLS)                                  | Robust regression (Huber/M-estimators), Theil-Sen, rank regression, permutation tests | `lm(y ~ x)`; `MASS::rlm()`; `mblm::mblm()`; permutation: `coin::symmetry_test()`                                                |      |
| **12**  | Goodness-of-fit (categorical)                               | Chi-square GOF / test of independence                   | Fisher’s exact (small n); permutation/exact tests                                     | `chisq.test(tbl)`; `fisher.test(tbl)`; `coin::chisq_test()`                                                                     |      |
| **13**  | Paired binary data                                          | —                                                       | McNemar test                                                                          | `mcnemar.test(matrix)`                                                                                                          |      |
| **14**  | Ordinal data (independent samples)                          | —                                                       | Mann–Whitney U / Kruskal–Wallis                                                       | `wilcox.test(x ~ g)`; `kruskal.test(y ~ g)`                                                                                     |      |
| **15**  | Ordinal data (paired samples)                               | —                                                       | Wilcoxon signed-rank                                                                  | `wilcox.test(x, y, paired=TRUE)`                                                                                                |      |
| **16**  | Normality / distribution check                              | Shapiro–Wilk, KS                                        | —                                                                                     | `shapiro.test(x)`; `ks.test(x, "pnorm", mean(x), sd(x))`                                                                        |      |
