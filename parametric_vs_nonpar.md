# Parametric vs Non-parametric / Ordinal / Paired Tests

| **No.** | **Purpose / Data Type**                      | **Parametric Test (Normal / Interval)** | **Non-parametric / Ordinal / Robust Alternative**                                     | **Notes**                                                                             | **R script example** |
| ------ | -------------------------------------------- | --------------------------------------- | ------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------- | --------------------- |
| 1 | **One-sample location (interval/ratio)**     | One-sample t-test                       | Wilcoxon signed-rank; Sign test                                                       | Wilcoxon assumes symmetric distribution; Sign test works for skewed data.             | `t.test(x, mu=0)`<br>`wilcox.test(x, mu=0)` |
| 2 | **Two independent samples (interval/ratio, equal variances)** | Independent t-test (student t)                      | Mann–Whitney U (Wilcoxon rank-sum); 2-sample KS                                       | Student t-test assumes equal variances; MW tests differences in central tendency (ranks). | `t.test(x, y, var.equal=TRUE)`<br>`wilcox.test(x, y)` |
| 3 | **Two independent samples (interval/ratio, unequal variances)** | Independent t-test (Welch’s t-test)                      | Mann–Whitney U (Wilcoxon rank-sum); 2-sample KS                                       | Welch adjusts degrees of freedom; use when group variances differ. MW tests differences in central tendency (ranks).        | `t.test(x, y)`<br>`wilcox.test(x, y)` |
| 4 | **Two paired samples (interval/ratio)**      | Paired t-test                           | Wilcoxon signed-rank; Sign test                                                       | Applied to paired differences.                                                        | `t.test(x, y, paired=TRUE)`<br>`wilcox.test(x, y, paired=TRUE)` |
| 5 | **≥3 independent groups (interval/ratio, equal variances)**   | Welch ANOVA --> followed by **Tukey** post-hoc                          | Kruskal–Wallis --> post-hoc: Dunn's Test                                                    | KW tests differences in medians/ranks.                                                | `oneway.test(y ~ g, var.equal=TRUE)`<br>`TukeyHSD(aov(y~g))`<br>`kruskal.test(y~g)`<br>`FSA::dunnTest(y~g)` |
| 6 | **≥3 independent groups (interval/ratio, unequal variances)**   | One-way ANOVA --> followed by **Games-Howell** post-hoc                          | Kruskal–Wallis; permutation ANOVA                                                     | KW tests differences in medians/ranks.                                                | `oneway.test(y ~ g)`<br>`userfriendlyscience::oneway(y, g, posthoc="games-howell")`<br>`kruskal.test(y~g)` |
| 7 | **≥3 related/repeated measures**             | Repeated-measures ANOVA                 | Friedman test                                                   | Friedman = rank version of repeated ANOVA.                                            | `aov(y ~ cond + Error(id/cond))`<br>`friedman.test(y ~ cond | id)` |
| 8 | **Two-way factorial (interval/ratio)**       | Two-way ANOVA                           | Aligned Rank Transform (ART); Scheirer–Ray–Hare (SRH); permutation factorial ANOVA    | ART handles interactions and main effects; SRH is older and limited for interactions. | `aov(y ~ A*B)`<br>`ARTool::art(y ~ A*B)` |
| 9 | **ANCOVA (factor(s) + covariate)**           | ANCOVA                                  | Quade’s test (rank ANCOVA); permutation ANCOVA; robust regression                     | Quade adjusts for covariate by ranking residual-like quantities.                      | `anova(lm(y ~ g + cov))`<br>`RVAideMemoire::quade.test(y, g, cov)` |
| 10 | **Correlation (interval/ratio)**             | Pearson correlation                     | Spearman rank correlation; Kendall’s tau                                              | Spearman/Kendall work for monotonic relationships, non-normal data.                   | `cor.test(x, y, method="pearson")`<br>`cor.test(x, y, method="spearman")` |
| 11 | **Regression coefficients (interval/ratio)** | t-test for slope (OLS)                  | Robust regression (Huber/M-estimators), Theil-Sen, rank regression, permutation tests | Robust methods reduce influence of outliers.                                          | `summary(lm(y ~ x))`<br>`MASS::rlm(y ~ x)`<br>`mblm::mblm(y~x)` |
| 12 | **Goodness-of-fit (categorical)**            | Chi-square GOF / test of independence   | Fisher’s exact (small n); permutation/exact tests                                     | Use Fisher for small cell counts.                                                     | `chisq.test(tbl)`<br>`fisher.test(tbl)` |
| 13 | **Paired binary data**                       | —                                       | McNemar test                                                                          | Tests change in paired dichotomous outcomes (e.g., pre/post).                         | `mcnemar.test(tbl)` |
| 14 | **Ordinal data (independent samples)**       | —                                       | Mann–Whitney U / Kruskal–Wallis                                                       | Ranks preserve order; do not assume equal spacing.                                    | `wilcox.test(x, y)`<br>`kruskal.test(y~g)` |
| 15 | **Ordinal data (paired samples)**            | —                                       | Wilcoxon signed-rank                                                                  | Tests for median shift in paired ordinal responses.                                   | `wilcox.test(x, y, paired=TRUE)` |
| 16 | **Normality / distribution check**           | Shapiro–Wilk, KS                        | —                                                                                     | If rejected, choose non-parametric alternatives above.                                | `shapiro.test(x)`<br>`ks.test(x, "pnorm", mean(x), sd(x))` |
