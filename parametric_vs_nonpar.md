# Parametric vs Non-parametric / Ordinal / Paired Tests

| **Purpose / Data Type**                      | **Parametric Test (Normal / Interval)** | **Non-parametric / Ordinal / Robust Alternative**                                     | **Notes**                                                                             |
| -------------------------------------------- | --------------------------------------- | ------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------- |
| **One-sample location (interval/ratio)**     | One-sample t-test                       | Wilcoxon signed-rank; Sign test                                                       | Wilcoxon assumes symmetric distribution; Sign test works for skewed data.             |
| **Two independent samples (interval/ratio, equal variances)** | Independent t-test                      | Mann–Whitney U (Wilcoxon rank-sum); 2-sample KS                                       | MW tests differences in central tendency (ranks).                                     |
| **Two paired samples (interval/ratio)**      | Paired t-test                           | Wilcoxon signed-rank; Sign test                                                       | Applied to paired differences.                                                        |
| **≥3 independent groups (interval/ratio)**   | One-way ANOVA                           | Kruskal–Wallis; permutation ANOVA                                                     | KW tests differences in medians/ranks.                                                |
| **≥3 related/repeated measures**             | Repeated-measures ANOVA                 | Friedman test; permutation methods                                                    | Friedman = rank version of repeated ANOVA.                                            |
| **Two-way factorial (interval/ratio)**       | Two-way ANOVA                           | Aligned Rank Transform (ART); Scheirer–Ray–Hare (SRH); permutation factorial ANOVA    | ART handles interactions and main effects; SRH is older and limited for interactions. |
| **ANCOVA (factor(s) + covariate)**           | ANCOVA                                  | Quade’s test (rank ANCOVA); permutation ANCOVA; robust regression                     | Quade adjusts for covariate by ranking residual-like quantities.                      |
| **Correlation (interval/ratio)**             | Pearson correlation                     | Spearman rank correlation; Kendall’s tau                                              | Spearman/Kendall work for monotonic relationships, non-normal data.                   |
| **Regression coefficients (interval/ratio)** | t-test for slope (OLS)                  | Robust regression (Huber/M-estimators), Theil-Sen, rank regression, permutation tests | Robust methods reduce influence of outliers.                                          |
| **Goodness-of-fit (categorical)**            | Chi-square GOF / test of independence   | Fisher’s exact (small n); permutation/exact tests                                     | Use Fisher for small cell counts.                                                     |
| **Paired binary data**                       | —                                       | McNemar test                                                                          | Tests change in paired dichotomous outcomes (e.g., pre/post).                         |
| **Ordinal data (independent samples)**       | —                                       | Mann–Whitney U / Kruskal–Wallis                                                       | Ranks preserve order; do not assume equal spacing.                                    |
| **Ordinal data (paired samples)**            | —                                       | Wilcoxon signed-rank                                                                  | Tests for median shift in paired ordinal responses.                                   |
| **Normality / distribution check**           | Shapiro–Wilk, KS                        | —                                                                                     | If rejected, choose non-parametric alternatives above.                                |
