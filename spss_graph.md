# Graph in SPSS

## 1. Diagnostic questions

1. **What are our variables?**
2. **Is the variable categorical or numerical?**
3. **What graph are you trying to make?**

Example:

| Variable | Type        |
| -------- | ----------- |
| Gender   | categorical |
| Age      | numerical   |
| Score    | numerical   |

This determines the correct graph.

---

## 2. The key concept:

> When making graphs with multiple variables, SPSS needs each variable to have a role:
>
> • one variable usually goes on the **X-axis**
> • one variable is the **outcome (Y-axis)**
> • another variable can **split the data into groups**

## 3. The most common “multi-variable” graphs in SPSS (know these!)

Students usually mean one of these:

### 1. Grouped bar chart

Example:

* X-axis: **Gender**
* Bars: **Average score**
* Groups: **School type**

**Steps in SPSS**

1. `Graphs`
2. `Chart Builder`
3. Choose **Bar**
4. Drag **Clustered Bar**
5. Put:

   * category variable → **x-axis**
   * outcome variable → **y-axis**
   * second variable → **Cluster on X / Group**

Explain simply:

> The X-axis shows one variable, and the second variable splits the bars into groups.

---

### 2. Line graph with groups

Example:

* X-axis: **Time**
* Y-axis: **Score**
* Lines: **Gender**

**Steps**

1. `Graphs`
2. `Chart Builder`
3. Choose **Line**
4. Drag **Multiple Line**
5. Assign:

   * X-axis variable
   * Y-axis variable
   * Group variable → **Set colour**

---

### 3. Boxplot comparing groups

Useful for:

* comparing distributions across groups.

Steps:

1. `Graphs`
2. `Chart Builder`
3. Choose **Boxplot**
4. Put:

   * scale variable → Y
   * group variable → X
